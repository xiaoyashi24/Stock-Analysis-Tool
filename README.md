# Interactive Monthly Stock Performance Analysis Tool

## Project Overview
This project develops a Python-based interactive data analysis tool for analysing monthly stock performance using data from the CSMAR database. The tool allows users to input a stock code and a starting month, retrieves monthly stock trading data, performs cleaning and transformation, and produces descriptive analysis, visualisations, and analytical insights.

## Analytical Problem
How can users quickly evaluate the monthly performance and volatility of a selected stock using Python and financial market data?

## Target Users
- business students
- finance learners
- beginner investors

## Dataset Information
- Source: CSMAR database
- Table used: `csmar.trd_mnth`

## Features
- user input of stock code and starting month
- data retrieval from database
- data cleaning and transformation
- descriptive analysis
- visualisation
- Excel export

## Tools and Libraries Used
- pandas
- numpy
- matplotlib
- os

## How to Run
1. Open the notebook.
2. Run the import and database connection cells.
3. Set the stock code and starting month.
4. Run all remaining cells in order.
5. Review outputs and charts.

## Output
- cleaned dataset
- summary statistics
- charts
- exported Excel file
## Files

- `Analysis Tool.ipynb`: main notebook for the interactive stock analysis tool
- `Analysis for 000001.ipynb`: example analysis for stock code 000001
- `README.md`: project description and usage instructions
## Conclusion
This project demonstrates how Python can be used to build a simple interactive stock analysis tool based on financial market data.
